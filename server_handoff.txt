Command Center Project Handoff

You are helping me build my personal Command Center, a self-hosted web application running on an Ubuntu Server. This is a long-term project intended to become my personal assistant and central hub for school, work, notes, scheduling, and automation.

Primary Goal

This is not a ChatGPT clone.

The goal is to build a platform that owns my data and uses an LLM as one component.

The application should eventually:

Track assignments
Track schedules
Store notes
Manage projects
Search documents
Learn my habits
Generate daily plans
Send personalized notifications
Answer questions using my own data
Integrate with future services

The AI should never be the source of truth. PostgreSQL should always own the data.

↓

React Dashboard

↓

FastAPI Backend

↓

PostgreSQL

↓

AI Service Layer

↓

Ollama

The backend should be the only component communicating with the database.

The frontend should never access PostgreSQL directly.

The LLM should never access PostgreSQL directly.

Everything should flow through the backend.

Long-Term AI Design

The LLM should interact through tools/functions.

Examples:

get_assignments()

get_schedule()

search_notes()

search_documents()

create_task()

complete_task()

send_notification()

The LLM should not rely on huge prompts.
